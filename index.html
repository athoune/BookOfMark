Book of Mark by Athoune

Write your doc in simple text files with markdown syntax. Build a PDF book and even print it, an eBook or a clean website.

Print it

Nobody can beats LaTeX. Your book can be LaTeX then PDF and be printed or read on a screen.

eRead it

Your book can be nice on a Kindle or any other eReader.

Version it

Your prose is plain old text, easily handled by any versioning tool, git, svn or even cvs.

Write it

Choose your weapon, TextWrangler, TextMate, Vim, Emacs, Bean, it's your choice.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/athoune/BookOfMark

get the source code on GitHub : athoune/BookOfMark
